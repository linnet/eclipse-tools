Quickmarks Plugin: What it does

This extension enables you to set, delete and jump to text markers by one key stroke only, without having to fill in dialog fields.

By default there are 10 Quickmarks per Workspace available, mapped to some key combination typically involving the keys 0-9 (default is Shift+Ctrl+0 - Shift+Ctrl+9 to set/delete/move a Quickmark and Alt+0 - Alt+9 to jump to the numbered Quickmark).

You can configure this plugin to behave differently via it's preferences. E.g. you can have a complete set of Quickmarks per project, folder or document, if you want; you can specify, how the selection is remebered when jumping to it, you can change the key binding and more.
